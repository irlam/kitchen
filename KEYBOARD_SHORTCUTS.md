# Keyboard Shortcuts Guide

## KitchenLab Pro - Quick Reference

### Item Management
- **Delete / Backspace**: Delete selected item
- **Escape**: Deselect current item
- **R**: Rotate selected item by 90 degrees

### View Controls
- **Ctrl/Cmd + 2**: Switch to 2D floorplan view
- **Ctrl/Cmd + 3**: Switch to 3D perspective view

### Project Management
- **Ctrl/Cmd + S**: Save current project

---

## Notes

- Keyboard shortcuts are disabled when typing in input fields or text areas
- Most shortcuts require an item to be selected to work
- Shortcuts work on Windows (Ctrl) and Mac (Cmd)

---

## Future Enhancements

Planned keyboard shortcuts for future releases:
- Arrow keys: Move selected item
- Ctrl/Cmd + Z: Undo last action
- Ctrl/Cmd + Y: Redo last action
- Ctrl/Cmd + D: Duplicate selected item
- Ctrl/Cmd + A: Select all items
- +/-: Zoom in/out
